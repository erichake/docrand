Erreur de paramètre dans l'URL
"<lien>" est l'adresse d'un fichier partagé en mode "accessible à tous les possesseurs du lien" sur un des clouds suivants :
  • NextCloud (adoptée par l'Education Nationale)
  • Google Drive
  • Dropbox
  • OneDrive
Note aux utilisateurs de Google Drive : le fichier ciblé est de type javascript, mais sur ce cloud vous devez éviter de choisir l'extension ".js" pour le nom de votre fichier. Choisissez plutôt l'extension ".txt". Pour les trois autres clouds, l'extension ".js" ne pose aucun problème particulier.
